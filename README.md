# Power_BI_Analyst_DIO
Formação Power BI para a criação de visualizações personalizadas e na utilização de recursos avançados com IA.


Além de aprender a coletar e transformar os dados utilizando os diversos recursos do Power BI. A Formação possui conteúdos sobre a integração com diferentes sistemas e modelagem de dados. E tudo isso com diversos projetos práticos sendo desenvolvidos durante a jornada, englobando todo o fluxo de análise de dados.
